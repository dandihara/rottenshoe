# rottenshoe

-> 로튼토마토를 모티브하여 만든 서비스입니다. 영화가 아닌 신발로 바꾸어서 사람들이 점수를 매기거나 sns형식의 칸을 두어 서로의 룩을 보는 컨텐츠와 
비슷한 유형의 신발을 추천 해 주는 것도 추가할 예정입니다.


-스니커즈 hot or cold 포인트 생성
=> 유행의 척도를 알 수 있게


-스니커즈 추천 시스템 추가 예정

- ajax 사용

- 게시판 형식

- create user 사용 => auth user model 재 설정

backend : django
frontend : vanilla js / html+css
